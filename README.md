## 👋 About Me

I'm a **backend software engineer** building production-grade systems in Java and Spring Boot — with a strong focus on security, reliability, and thoughtful system design.

- 🔐 Just shipped **Insighta Labs+** — GitHub OAuth + PKCE, JWT lifecycle management, role-based access, CLI tool published to npm, web portal on Netlify, backend on Google Cloud Run
- 🎬 Shipped **Cinema Booking App** — full payment lifecycle with Paystack webhooks, HMAC verification, idempotency, and seat lock scheduling
- 💰 Shipped **Budget App** — Spring Security, JWT, BCrypt, 20+ integration tests, deployed on Render
- 📐 Design before code — always start with diagrams, failure modes, and security considerations
- 🤝 Open to backend engineering roles — especially fintech, payments, and AI infrastructure

## 🧠 Engineering Principles I Live By

```
1. Bad path before happy path — think about failure before success.
2. Security lens on everything — userId from JWT, never from request headers.
3. Understand the problem before touching the keyboard.
4. Readable code > clever code. The next person reading it might be you.
5. Ship small, iterate fast, learn constantly.
6. Great software is 20% code, 80% decisions.
```
